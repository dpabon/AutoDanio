## AutoDanio##

Programa creado con el fin de modelar el patrón de coloración de _Danio_ _rerio_ en varias etapas de desarrollo.

### Dependencias requeridas ###

1. Python
2. Numpy
3. Python-OpenCV
5. Ipython Notebook
